## Чтение книг

### Базовое чтение FB2

```java
import org.tehlab.whitek0t.fictionbook.api.FictionBookIO;
import org.tehlab.whitek0t.fictionbook.dto.FictionBookDto;

FictionBookDto book = FictionBookIO.read(Path.of("war_and_peace.fb2"));

// Метаданные
String title = book.description().titleInfo().bookTitle();
List<Author> authors = book.description().titleInfo().authors();
List<String> genres = book.description().titleInfo().genres();

System.out.println("Название: " + title);
authors.forEach(a -> System.out.println("Автор: " + a.getFullName()));
```

### Чтение из InputStream

```java
try (InputStream is = new FileInputStream("book.fb2")) {
    Fb2Reader reader = new Fb2Reader();
    FictionBookDto book = reader.read(is);
    // ...
}
```

### Автоопределение формата (FB2/FB3)

```java
Path file = Path.of("book.unknown");
FictionBookFormat format = FictionBookFormat.detect(file);

System.out.println("Формат: " + format); // FB2 или FB3
FictionBookDto book = FictionBookIO.read(file); // Сам выберет нужный reader
```

### Доступ к содержимому

```java
FictionBookDto book = FictionBookIO.read(file);

// Обход всех секций
for (BodyDto body : book.bodies()) {
    System.out.println("Body: " + (body.name() != null ? body.name() : "main"));
    
    for (Section section : body.sections()) {
        System.out.println("  Section: " + section.id());
        
        for (BlockElement block : section.content()) {
            if (block instanceof Paragraph p) {
                // Обработка параграфа
            } else if (block instanceof Poem poem) {
                // Обработка стихотворения
            }
        }
    }
}
```

### Доступ к ресурсам (картинкам)

```java
FictionBookDto book = FictionBookIO.read(file);

// Список всех ресурсов
for (Resource resource : book.resources().values()) {
    System.out.println("ID: " + resource.id());
    System.out.println("Type: " + resource.contentType());
    
    // Чтение данных (ленивое)
    try (InputStream is = resource.dataProvider().getInputStream()) {
        byte[] bytes = is.readAllBytes();
        Files.write(Path.of("extracted_" + resource.id() + ".jpg"), bytes);
    }
}
```

## Потоковое чтение (Streaming)

`FictionBookStreamer` читает книгу **посекционно**, не удерживая всё дерево в памяти —
это нужно читалкам и обработке больших файлов. Формат определяется автоматически:
`open()` возвращает `Fb2Streamer` для FB2 или `Fb3Streamer` для FB3. Общая гарантия —
*ленивые секции*: `readDescription()` парсит метаданные лениво и кэширует;
`readNextSection()` отдаёт по одной секции верхнего уровня; `getResource(id)` резолвит
картинки по требованию (он сам срезает ведущий `#`).

> ℹ️ FB2 догружает бинарники из конца файла по требованию; FB3 — это OPC/ZIP с
> дефлейтом, поэтому контейнер распаковывается в память целиком (seek по нему
> невозможен), но дерево секций всё равно строится лениво. Секции тела сносок
> (`<body name="notes">` / FB3 `notes.xml`) идут после основных в одном потоке.

### Поглавное чтение

```java
import org.tehlab.whitek0t.fictionbook.api.FictionBookStreamer;

try (FictionBookStreamer streamer = FictionBookStreamer.open(Path.of("big_book.fb2"))) {

    // 1. Метаданные (быстро, без загрузки тела)
    Description desc = streamer.readDescription();
    System.out.println("Книга: " + desc.titleInfo().bookTitle());

    // 2. Секции по одной — предыдущие не держатся в памяти
    Section section;
    while ((section = streamer.readNextSection()) != null) {
        System.out.println("Секция: " + section.id());
        // ... отрисовать главу в читалке и забыть про неё ...
    }
}
```

### Рендеринг глав на лету (с картинками)

```java
try (FictionBookStreamer streamer = FictionBookStreamer.open(path)) {
    streamer.readDescription();

    PlainTextRenderer renderer = new PlainTextRenderer();
    // Картинки резолвятся лениво через getResource (он сам срезает ведущий '#').
    // getResource — checked-метод, поэтому оборачиваем в лямбду.
    BookPlayer player = new BookPlayer(renderer, href -> {
        try {
            return streamer.getResource(href);
        } catch (FictionBookException e) {
            return null;
        }
    });

    Section section;
    while ((section = streamer.readNextSection()) != null) {
        player.playSection(section);
    }
    System.out.println(renderer.getOutput());
}
```

### Индекс якорей по стримеру

```java
try (FictionBookStreamer streamer = FictionBookStreamer.open(path)) {
    // ⚠️ buildAnchorIndex читает книгу целиком — это НЕ потоковая операция
    // (индекс якорей по природе охватывает весь файл).
    AnchorIndex index = streamer.buildAnchorIndex();
    System.out.println("Якорей: " + index.size());
}
```

## Краткая сводка о книге (BookInfo)

`FictionBookIO.info(...)` собирает готовую к показу сводку: метаданные, объём в
символах/словах, обложку, человекочитаемые жанры и внешние ссылки. Это удобнее
ручного обхода `description` для каталогов и карточек книг.

```java
import org.tehlab.whitek0t.fictionbook.api.BookInfo;

// Из файла (имя файла попадёт в сводку) ...
BookInfo info = FictionBookIO.info(Path.of("book.fb2"));
// ... или из уже прочитанного DTO:
BookInfo info2 = FictionBookIO.info(book);

System.out.println(info.title());          // название
System.out.println(info.authorsLine());    // "Юрий Винокуров, Олег Сапфир"
System.out.println(info.genreNames());     // ["ЛитРПГ"] — коды → русские названия
System.out.println(info.wordCount());      // объём в словах
System.out.println(info.hasCover());       // есть ли обложка
System.out.println(info.links());          // только внешние ссылки (без якорей '#')

// Готовое многострочное представление (пустые поля опускаются):
System.out.println(info.toDisplayString());
```

## Рендеринг в HTML

### Полный HTML-документ с base64-картинками

```java
FictionBookDto book = FictionBookIO.read(Path.of("book.fb2"));

HtmlRenderer renderer = HtmlRenderer.builder()
    .wrapInHtmlDocument(true)
    .title(book.description().titleInfo().bookTitle())
    .resourceResolver(ResourceResolver.base64DataUri())  // Картинки внутри HTML
    .build();

BookPlayer player = new BookPlayer(renderer, 
    href -> resolveResource(book, href));
player.play(book);

Files.writeString(Path.of("book.html"), renderer.getOutput());
```

### HTML с сохранением картинок в папку

```java
Path imagesDir = Path.of("output/images");
Files.createDirectories(imagesDir);

HtmlRenderer renderer = HtmlRenderer.builder()
    .wrapInHtmlDocument(true)
    .title("Моя книга")
    .resourceResolver(ResourceResolver.saveToDirectory(imagesDir, "images/"))
    .build();

BookPlayer player = new BookPlayer(renderer, 
    href -> resolveResource(book, href));
player.play(book);

Files.writeString(Path.of("output/book.html"), renderer.getOutput());
```

### HTML-фрагмент (для встраивания в шаблон)

```java
HtmlRenderer renderer = HtmlRenderer.builder()
    .wrapInHtmlDocument(false)  // Только <section>...</section>
    .resourceResolver(ResourceResolver.base64DataUri())
    .build();

new BookPlayer(renderer).play(book);

String fragment = renderer.getOutput();
// Встраиваем fragment в Thymeleaf/FreeMarker/другой шаблонизатор
```

### HTML с кастомным CSS

```java
String myCss = """
    body { font-family: Arial; background: #f0f0f0; }
    .section-title { color: navy; }
    """;

HtmlRenderer renderer = HtmlRenderer.builder()
    .wrapInHtmlDocument(true)
    .title(book.description().titleInfo().bookTitle())
    .customCss(myCss)  // Добавляется после стандартного CSS
    .resourceResolver(ResourceResolver.base64DataUri())
    .build();
```

### Вспомогательный метод для резолва ресурсов

```java
private static Resource resolveResource(FictionBookDto book, String href) {
    if (href == null) return null;
    
    // Убираем # из внутренних ссылок
    String id = href.startsWith("#") ? href.substring(1) : href;
    
    // Убираем имя файла из ссылок типа "notes.xml#n1"
    int hashIdx = id.indexOf('#');
    if (hashIdx > 0) {
        id = id.substring(hashIdx + 1);
    }
    
    return book.resources().get(id);
}
```

## Рендеринг в Markdown

```java
FictionBookDto book = FictionBookIO.read(file);

// Резолвер картинок: base64 data-URI (или saveToDirectory / null для placeholder)
MarkdownRenderer renderer = new MarkdownRenderer(ResourceResolver.base64DataUri());

BookPlayer player = new BookPlayer(renderer, href -> lookupResource(book, href));
player.play(book);

Files.writeString(Path.of("book.md"), renderer.getOutput());
```

Сопоставление: заголовки секций → `#`…`######` (по глубине вложенности),
`**жирный**`/`*курсив*`/`~~зачёркнутый~~`, цитаты/эпиграфы → `> blockquote`,
таблицы → GFM-таблицы, картинки → `![alt](src)`, ссылки → `[текст](href)`.
Без резолвера (`new MarkdownRenderer()`) картинки становятся текстовым
placeholder'ом `*(изображение: alt)*`.

## CLI: конвертация в txt, html и md

В проекте есть тонкая утилита-приложение `cli/FictionBookCli` — конвертер
FB2/FB3 → `txt`/`html`/`md`. Она только связывает публичные фасады (`FictionBookIO`,
`BookInfo`, `BookPlayer`, рендереры) и не добавляет своей логики разбора/рендеринга.
Перед телом выводит шапку из метаданных: название, авторов, обложку и аннотацию.

### Сборка и запуск

```bash
# Прямой запуск через Gradle (аргументы — в --args)
./gradlew run --args="book.fb2 -f html"

# Готовый дистрибутив с launcher'ом (build/install/fb/bin/fb)
./gradlew installDist
build/install/fb/bin/fb book.fb2 -f html
```

### Примеры

```bash
# FB2 -> текст (по умолчанию рядом с исходником: book.txt)
fb book.fb2

# FB2 -> HTML со встроенными (base64) картинками
fb book.fb2 -f html

# Явный выход; формат выводится из расширения (.html)
fb book.fb2 -o out/book.html

# Извлечь картинки в отдельную папку out/book_files/
fb book.fb2 -o out/book.html --images extract

# Только HTML-фрагмент (без <html>/<head>/<body>)
fb book.fb2 -f html --no-wrap -o fragment.html

# FB2 -> Markdown (GFM); картинки извлекаются в out/book_files/
fb book.fb2 -f md -o out/book.md --images extract

# Markdown в stdout без картинок (placeholder вместо них)
fb book.fb2 -f md --images none -o -

# В stdout — удобно для пайпов и грепа
fb book.fb2 -o - | grep "Глава"
```

### Опции

| Опция | Значение | По умолчанию |
|-------|----------|--------------|
| `-f`, `--format` | `txt` \| `html` \| `md` | по расширению выхода (`.html`/`.md`), иначе `txt` |
| `-o`, `--output` | путь (`-` → stdout) | рядом с входом, новое расширение |
| `--images` | `embed` (base64) \| `extract` (папка `<имя>_files`) \| `none` | `embed` |
| `--no-wrap` | HTML-фрагмент без обёртки документа | обёртка включена |
| `-h`, `--help` / `-v`, `--version` | справка / версия | — |

> **Картинки.** Конвертируются и инлайновые `<image>` внутри `<p>`, и блочные
> `<image>` (прямой ребёнок `<section>` — типичная иллюстрация главы). В `md`/`html`
> режим `embed` встраивает их base64, `extract` — сохраняет файлами, `none` —
> placeholder (в `md` это текст `*(изображение: …)*`). При выводе в stdout режим
> `extract` деградирует до `embed` (некуда складывать файлы).

### Кастомный пайплайн


### Применение одного санитайзера

```java

```

### Создание собственного санитайзера

```java
public class SpoilerCleaner implements Sanitizer {
    @Override
    public FictionBookDto sanitize(FictionBookDto book) {
        return FictionBookDtoTransformer.transform(book)
            .onParagraph(p -> {
                String text = extractText(p);
                return text.contains("СПОЙЛЕР") ? null : p;
            })
            .apply();
    }
    
    private String extractText(Paragraph p) {
        return p.elements().stream()
            .filter(e -> e instanceof Text)
            .map(e -> ((Text) e).value())
            .collect(Collectors.joining());
    }
}

// Использование
SanitizerPipeline pipeline = SanitizerPipeline.builder()
    .add(new EmptyParagraphCleaner())
    .add(new SpoilerCleaner())  // Кастомный
    .add(new TextNodeMerger())
    .build();
```

### Проверка изменений после санитизации

```java
FictionBookDto before = book;
FictionBookDto after = SanitizerPipeline.standard().sanitize(book);

long paragraphsBefore = countParagraphs(before);
long paragraphsAfter = countParagraphs(after);

System.out.println("Удалено параграфов: " + (paragraphsBefore - paragraphsAfter));
```

```java
AnchorIndex index = AnchorIndexBuilder.fromDto(book);

String href = "#note_42";
if (index.canResolve(href)) {
```

### Валидация всех ссылок в книге

```java
AnchorIndex index = AnchorIndexBuilder.fromDto(book);
List<String> brokenLinks = new ArrayList<>();

// Обход всех Link в книге (через FictionBookDtoTransformer)
FictionBookDtoTransformer.transform(book)
    .onInlineElement(inline -> {

                && link.href().startsWith("#")) {

                brokenLinks.add(link.href());
            }
        }
        return inline;
    })
    .apply();

System.out.println("Битых ссылок: " + brokenLinks.size());
brokenLinks.forEach(System.out.println);
```

## Конвертация между форматами

### FB2 → HTML (полный конвертер)

```java
public class Fb2ToHtmlConverter {
    
    public void convert(Path fb2File, Path outputDir) throws Exception {
        FictionBookDto book = FictionBookIO.read(fb2File);
        
        // Создаём папку для картинок
        Path imagesDir = outputDir.resolve("images");
        Files.createDirectories(imagesDir);
        
        // Рендерер
        HtmlRenderer renderer = HtmlRenderer.builder()
            .wrapInHtmlDocument(true)
            .title(book.description().titleInfo().bookTitle())
            .resourceResolver(ResourceResolver.saveToDirectory(imagesDir, "images/"))
            .build();
        
        // Проигрыватель
        BookPlayer player = new BookPlayer(renderer, 
            href -> resolveResource(book, href));
        player.play(book);
        
        // Сохраняем
        String baseName = fb2File.getFileName().toString()
            .replace(".fb2", "");
        Path htmlFile = outputDir.resolve(baseName + ".html");
        Files.writeString(htmlFile, renderer.getOutput());
    }
}
```

### FB2 → TXT

```java

public void convertDirectory(Path inputDir, Path outputDir) throws Exception {
    try (var executor = Executors.newVirtualThreadPerTaskExecutor()) {
        Files.list(inputDir)
            .filter(p -> p.toString().endsWith(".fb2"))
            .forEach(fb2File -> executor.submit(() -> {
                try {
                    new Fb2ToHtmlConverter().convert(fb2File, outputDir);
                    System.out.println("Converted: " + fb2File.getFileName());
                } catch (Exception e) {
                    System.err.println("Failed: " + fb2File.getFileName() + " - " + e);
                }
            }));
    }
}
```

### Генерация каталога библиотеки

> Готовую сводку проще получить через `FictionBookIO.info(file)` (см.
> [Краткая сводка о книге](#краткая-сводка-о-книге-bookinfo)). Ниже — вариант с
> превью из `PlainTextRenderer`, которого нет в `BookInfo`; своя запись названа
> `CatalogEntry`, чтобы не путать с публичным `api.BookInfo`.

```java
public record CatalogEntry(
    String title,
    List<String> authors,
    List<String> genres,
    String language,
    String preview,
    long wordCount
) {}

public List<CatalogEntry> buildCatalog(Path libraryDir) throws Exception {
    List<CatalogEntry> catalog = new ArrayList<>();
    
    try (DirectoryStream<Path> stream = Files.newDirectoryStream(libraryDir, "*.fb2")) {
        for (Path file : stream) {
            try {
                FictionBookDto book = FictionBookIO.read(file);
                
                PlainTextRenderer renderer = new PlainTextRenderer();
                new BookPlayer(renderer).play(book);
                
                catalog.add(new CatalogEntry(
                    book.description().titleInfo().bookTitle(),
                    book.description().titleInfo().authors().stream()
                        .map(Author::getFullName).toList(),
                    book.description().titleInfo().genres(),
                    book.description().titleInfo().lang(),
                    renderer.getPreview(300),
                    renderer.getWordCount()
                ));
            } catch (Exception e) {
                System.err.println("Skipping " + file + ": " + e.getMessage());
            }
        }
    }
    
    return catalog;
}
```

### Поиск дубликатов по названию и автору

```java
public Map<String, List<Path>> findDuplicates(Path libraryDir) throws Exception {
    Map<String, List<Path>> duplicates = new HashMap<>();
    
    try (DirectoryStream<Path> stream = Files.newDirectoryStream(libraryDir, "*.fb2")) {
        for (Path file : stream) {
            try {
                FictionBookDto book = FictionBookIO.read(file);
                String key = normalizeKey(book);
                duplicates.computeIfAbsent(key, k -> new ArrayList<>()).add(file);
            } catch (Exception ignored) {}
        }
    }
    
    // Оставляем только реальные дубликаты (>1 файла)
    return duplicates.entrySet().stream()
        .filter(e -> e.getValue().size() > 1)
        .collect(Collectors.toMap(Map.Entry::getKey, Map.Entry::getValue));
}

private String normalizeKey(FictionBookDto book) {
    String title = book.description().titleInfo().bookTitle()
        .toLowerCase().replaceAll("\\s+", " ").trim();
    String authors = book.description().titleInfo().authors().stream()
        .map(Author::getFullName)
        .sorted()
        .collect(Collectors.joining(";"))
        .toLowerCase();
    return title + "|" + authors;
}
```

## Обработка ошибок

### Правильная обработка исключений

```java
try {
    FictionBookDto book = FictionBookIO.read(Path.of("book.fb2"));
    // ...
} catch (InvalidFormatException e) {
    System.err.println("Невалидный FB2: " + e.getMessage());
    System.err.println("  Файл: " + e.getFileName());
    System.err.println("  Строка: " + e.getLine());
    System.err.println("  Элемент: " + e.getElementName());
} catch (FictionBookException e) {
    System.err.println("Ошибка чтения: " + e.getMessage());
}
```

### Пропуск проблемных файлов при пакетной обработке

```java
for (Path file : files) {
    try {
        FictionBookDto book = FictionBookIO.read(file);
        process(book);
    } catch (InvalidFormatException e) {
        log.warn("Invalid FB2 format in {}: {}", file, e.getMessage());
        // Продолжаем со следующим файлом
    } catch (FictionBookException e) {
        log.error("Failed to read {}: {}", file, e.getMessage());
    }
}
```

## Советы и лучшие практики

### ✅ Используйте FictionBookIO для простоты

```java
// ✅ Хорошо
FictionBookDto book = FictionBookIO.read(file);
FictionBookIO.write(book, outputFile);

// ❌ Избыточно для большинства случаев
Fb2Reader reader = new Fb2Reader();
FictionBookDto book = reader.read(file);
```

### ✅ Переиспользуйте рендереры

```java
// ✅ Хорошо — рендерер потокобезопасен (если не используется параллельно)
HtmlRenderer renderer = HtmlRenderer.builder()
    .resourceResolver(ResourceResolver.base64DataUri())
    .build();

for (FictionBookDto book : books) {
    BookPlayer player = new BookPlayer(renderer);
    player.play(book);
    // ...
}
```

### ✅ Применяйте санитайзеры перед записью

```java
// ✅ Хорошо — стандартный пайплайн
FictionBookIO.write(book, outputFile);

// ⚠️ Опасно — можно записать "грязный" FB2
Fb2Writer writer = new Fb2Writer();
writer.setSanitizerPipeline(null);
writer.write(book, outputFile);
```

### ✅ Используйте ResourceResolver для управления картинками

```java
// Для маленьких книг — base64
ResourceResolver.base64DataUri()

// Для больших книг — сохранение в папку
ResourceResolver.saveToDirectory(imagesDir, "images/")

// Для превью/отладки — placeholder
ResourceResolver.placeholder()
```

### ⚠️ Не модифицируйте DTO напрямую

DTO immutable — для "изменения" создавайте новую копию:

```java
// ✅ Правильно
FictionBookDtoTransformer.transform(book)
    .onParagraph(p -> /* ... */)
    .apply();

// ❌ Невозможно — DTO на Records
```
