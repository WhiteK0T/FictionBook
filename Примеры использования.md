## Чтение книг

### Базовое чтение FB2

```java
import org.tehlab.whitek0t.fictionbook.api.FictionBookIO;
import org.tehlab.whitek0t.fictionbook.dto.FictionBookDto;

FictionBookDto book = FictionBookIO.read(Path.of("war_and_peace.fb2"));

// Метаданные
String title = book.description().titleInfo().bookTitle();
List<Author> authors = book.description().titleInfo().authors();
List<String> genres = book.description().titleInfo().genres();

System.out.println("Название: " + title);
authors.forEach(a -> System.out.println("Автор: " + a.getFullName()));
```

```java

```

### Доступ к ресурсам (картинкам)

```java

FictionBookDto book = FictionBookIO.read(Path.of("input.fb2"));

// Запись с автоматической санитизацией
```

```java
Fb2Writer writer = new Fb2Writer();
writer.setPrettyPrint(true);  // Форматированный вывод

```

### Отключение санитизации

```java
Fb2Writer writer = new Fb2Writer();
writer.setSanitizerPipeline(null);  // Записываем "как есть"
writer.write(book, Path.of("raw_output.fb2"));
```

## Рендеринг в HTML

### Полный HTML-документ с base64-картинками

```java
FictionBookDto book = FictionBookIO.read(Path.of("book.fb2"));

HtmlRenderer renderer = HtmlRenderer.builder()
    .wrapInHtmlDocument(true)
    .title(book.description().titleInfo().bookTitle())
    .resourceResolver(ResourceResolver.base64DataUri())  // Картинки внутри HTML
    .build();

BookPlayer player = new BookPlayer(renderer, 
    href -> resolveResource(book, href));
player.play(book);

Files.writeString(Path.of("book.html"), renderer.getOutput());
```

### HTML с сохранением картинок в папку

```java
Path imagesDir = Path.of("output/images");
Files.createDirectories(imagesDir);

HtmlRenderer renderer = HtmlRenderer.builder()
    .wrapInHtmlDocument(true)
    .title("Моя книга")
    .resourceResolver(ResourceResolver.saveToDirectory(imagesDir, "images/"))
    .build();

BookPlayer player = new BookPlayer(renderer, 
    href -> resolveResource(book, href));
player.play(book);

```

### HTML-фрагмент (для встраивания в шаблон)

```java

    .build();

```

### HTML с кастомным CSS

```java
String myCss = """
    body { font-family: Arial; background: #f0f0f0; }

    """;

HtmlRenderer renderer = HtmlRenderer.builder()
    .wrapInHtmlDocument(true)
```

### Вспомогательный метод для резолва ресурсов

```java
private static Resource resolveResource(FictionBookDto book, String href) {
    if (href == null) return null;
    
    // Убираем # из внутренних ссылок
    String id = href.startsWith("#") ? href.substring(1) : href;
    
    // Убираем имя файла из ссылок типа "notes.xml#n1"
    int hashIdx = id.indexOf('#');
    if (hashIdx > 0) {
        id = id.substring(hashIdx + 1);
    }
    
    return book.resources().get(id);
}
```

### Создание собственного санитайзера

```java
public class SpoilerCleaner implements Sanitizer {
    @Override
    public FictionBookDto sanitize(FictionBookDto book) {
        return FictionBookDtoTransformer.transform(book)
            .onParagraph(p -> {
                String text = extractText(p);
                return text.contains("СПОЙЛЕР") ? null : p;
            })
            .apply();
    }
    
    private String extractText(Paragraph p) {
        return p.elements().stream()
            .filter(e -> e instanceof Text)
            .map(e -> ((Text) e).value())
            .collect(Collectors.joining());
    }
}

// Использование
SanitizerPipeline pipeline = SanitizerPipeline.builder()
    .add(new EmptyParagraphCleaner())
    .add(new SpoilerCleaner())  // Кастомный
    .add(new TextNodeMerger())
    .build();
```

### Проверка изменений после санитизации

### Валидация всех ссылок в книге

```java
AnchorIndex index = AnchorIndexBuilder.fromDto(book);
List<String> brokenLinks = new ArrayList<>();

// Обход всех Link в книге (через FictionBookDtoTransformer)
FictionBookDtoTransformer.transform(book)
    .onInlineElement(inline -> {
        if (inline instanceof Link link && link.href() != null 
                && link.href().startsWith("#")) {

                brokenLinks.add(link.href());
            }
        }
        return inline;
    })
    .apply();
```

## Конвертация между форматами

### FB2 → HTML (полный конвертер)

```java
public class Fb2ToHtmlConverter {
    
    public void convert(Path fb2File, Path outputDir) throws Exception {
        FictionBookDto book = FictionBookIO.read(fb2File);
        
        // Создаём папку для картинок
        Path imagesDir = outputDir.resolve("images");
        Files.createDirectories(imagesDir);
        
        // Рендерер
        HtmlRenderer renderer = HtmlRenderer.builder()
            .wrapInHtmlDocument(true)
            .title(book.description().titleInfo().bookTitle())
            .resourceResolver(ResourceResolver.saveToDirectory(imagesDir, "images/"))
            .build();
        
        // Проигрыватель
        BookPlayer player = new BookPlayer(renderer, 
            href -> resolveResource(book, href));
        player.play(book);
        
        // Сохраняем
        String baseName = fb2File.getFileName().toString()
            .replace(".fb2", "");
        Path htmlFile = outputDir.resolve(baseName + ".html");
        Files.writeString(htmlFile, renderer.getOutput());
    }
}
```

### FB2 → TXT

## Пакетная обработка

### Конвертация папки с FB2 в HTML

```java
public void convertDirectory(Path inputDir, Path outputDir) throws Exception {
    try (var executor = Executors.newVirtualThreadPerTaskExecutor()) {
        Files.list(inputDir)
            .filter(p -> p.toString().endsWith(".fb2"))
            .forEach(fb2File -> executor.submit(() -> {
                try {
                    new Fb2ToHtmlConverter().convert(fb2File, outputDir);
                    System.out.println("Converted: " + fb2File.getFileName());
                } catch (Exception e) {
                    System.err.println("Failed: " + fb2File.getFileName() + " - " + e);
                }
            }));
    }
}
```

### Генерация каталога библиотеки

```java
public record BookInfo(
    String title,
    List<String> authors,
    List<String> genres,
    String language,
    String preview,
    long wordCount
) {}

public List<BookInfo> buildCatalog(Path libraryDir) throws Exception {
    List<BookInfo> catalog = new ArrayList<>();
    
    try (DirectoryStream<Path> stream = Files.newDirectoryStream(libraryDir, "*.fb2")) {
        for (Path file : stream) {
            try {
                FictionBookDto book = FictionBookIO.read(file);
                
                PlainTextRenderer renderer = new PlainTextRenderer();
                new BookPlayer(renderer).play(book);
                
                catalog.add(new BookInfo(
                    book.description().titleInfo().bookTitle(),
                    book.description().titleInfo().authors().stream()
                        .map(Author::getFullName).toList(),
                    book.description().titleInfo().genres(),
                    book.description().titleInfo().lang(),
                    renderer.getPreview(300),
                    renderer.getWordCount()
                ));
            } catch (Exception e) {
                System.err.println("Skipping " + file + ": " + e.getMessage());
            }
        }
    }
    
    return catalog;
}
```

### Поиск дубликатов по названию и автору

```java
public Map<String, List<Path>> findDuplicates(Path libraryDir) throws Exception {
    Map<String, List<Path>> duplicates = new HashMap<>();
    
    try (DirectoryStream<Path> stream = Files.newDirectoryStream(libraryDir, "*.fb2")) {
        for (Path file : stream) {
            try {
                FictionBookDto book = FictionBookIO.read(file);
                String key = normalizeKey(book);
                duplicates.computeIfAbsent(key, k -> new ArrayList<>()).add(file);
            } catch (Exception ignored) {}
        }
    }
    
    // Оставляем только реальные дубликаты (>1 файла)
    return duplicates.entrySet().stream()
        .filter(e -> e.getValue().size() > 1)
        .collect(Collectors.toMap(Map.Entry::getKey, Map.Entry::getValue));
}

private String normalizeKey(FictionBookDto book) {
    String title = book.description().titleInfo().bookTitle()
        .toLowerCase().replaceAll("\\s+", " ").trim();
    String authors = book.description().titleInfo().authors().stream()
        .map(Author::getFullName)
        .sorted()
        .collect(Collectors.joining(";"))
        .toLowerCase();
    return title + "|" + authors;
}
```

## Обработка ошибок

### Правильная обработка исключений

```java
try {
    FictionBookDto book = FictionBookIO.read(Path.of("book.fb2"));
    // ...
} catch (InvalidFormatException e) {
    System.err.println("Невалидный FB2: " + e.getMessage());
    System.err.println("  Файл: " + e.getFileName());
    System.err.println("  Строка: " + e.getLine());

    System.err.println("Ошибка чтения: " + e.getMessage());
}
```

### Пропуск проблемных файлов при пакетной обработке

```java
for (Path file : files) {
    try {
        FictionBookDto book = FictionBookIO.read(file);
        process(book);
    } catch (InvalidFormatException e) {
        log.warn("Invalid FB2 format in {}: {}", file, e.getMessage());
        // Продолжаем со следующим файлом
    } catch (FictionBookException e) {
        log.error("Failed to read {}: {}", file, e.getMessage());
    }
}
```

## Советы и лучшие практики

### ✅ Используйте FictionBookIO для простоты

```java
// ✅ Хорошо
FictionBookDto book = FictionBookIO.read(file);
FictionBookIO.write(book, outputFile);

// ❌ Избыточно для большинства случаев
Fb2Reader reader = new Fb2Reader();
FictionBookDto book = reader.read(file);
```

### ✅ Переиспользуйте рендереры

```java
// ✅ Хорошо — рендерер потокобезопасен (если не используется параллельно)
HtmlRenderer renderer = HtmlRenderer.builder()
    .resourceResolver(ResourceResolver.base64DataUri())
    .build();

for (FictionBookDto book : books) {
    BookPlayer player = new BookPlayer(renderer);
    player.play(book);
    // ...
}
```

### ✅ Применяйте санитайзеры перед записью

```java
// ✅ Хорошо — стандартный пайплайн
FictionBookIO.write(book, outputFile);

// ⚠️ Опасно — можно записать "грязный" FB2
Fb2Writer writer = new Fb2Writer();
writer.setSanitizerPipeline(null);
writer.write(book, outputFile);
```

### ✅ Используйте ResourceResolver для управления картинками

```java
// Для маленьких книг — base64
ResourceResolver.base64DataUri()

// Для больших книг — сохранение в папку
ResourceResolver.saveToDirectory(imagesDir, "images/")

// Для превью/отладки — placeholder
ResourceResolver.placeholder()
```

### ⚠️ Не модифицируйте DTO напрямую

DTO immutable — для "изменения" создавайте новую копию:

```java
// ✅ Правильно
FictionBookDtoTransformer.transform(book)
    .onParagraph(p -> /* ... */)
    .apply();

// ❌ Невозможно — DTO на Records
```
